• First, make sure the HTTP API in your Nuki Bridge is enabled. You can do this within the Nuki smartphone app. Go to "Manage my devices" and select "Bridge". There you can enable the HTTP API.
  • Once enabled, continue this procedure by tapping the button "Next >".
  • When the discovery process is running, press the button of your Nuki Bridge.
  • Your Nuki devices will be discovered and you can complete the pairing procedure.
